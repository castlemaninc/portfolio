About Me

I'm Bryan Castleman. I am a Front-End Developer. I became interested in Web Development through Graphic Design & Web Design using Adobe Software. In my experience I found that I liked the user experience and functionality of the web more than design in print. I wanted to be able to learn how to write my own code in languages such as HTML, CSS, and JavaScript so that I was not reliant on WYSIWYG software.

I've spent time building a strong foundation in HTML, CSS, and "vanilla" JavaScript. I began my studies in code using online resources such as Codecademy, Treehouse, freeCodeCamp, and Coursera, etc. I then decided it would be good to speed up the learning process by attending a coding bootcamp.

I succesfully completed a 6 month program at The Coding Bootcamp at UCLA Extension in 2017. I also have a BA in Art from CSULB in 2006.

I recently have been learning about CSS Grid, Flexbox, and "higher-order functions" in JavaScript. I'm also learning the ins and outs of Wordpress development with PHP.
